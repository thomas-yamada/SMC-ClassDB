# Semester and Classes Data Collector

This code is Step #1 for a dynamic class occupation and private room appointment visual display website. The goal is to provide convience to the students of Santa Monica College when looking for any public information reguaring any class.

The idea for the data collection originally came from [Kevin's](https://github.com/obscuradesign) project [Apollo](https://github.com/obscuradesign/apollo).
